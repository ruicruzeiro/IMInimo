Política de Privacidade


Data da Última Actualização
Esta Política de Privacidade foi actualizada pela última vez em 10-11-2024.

Identificação e Contacto do Responsável pelo Tratamento de Dados
Este site realiza o tratamento de dados pessoais exclusivamente para fins de cálculo imediato, descartando-os logo após o processamento, sem qualquer armazenamento contínuo. Para garantir a conformidade com as normas de protecção de dados, um responsável pela proteção de dados (RPD) foi designado e está disponível para esclarecer dúvidas ou tratar solicitações relacionadas com a privacidade e segurança dos dados fornecidos. Poderá contactar-nos para este efeito através do endereço geral@iminimo.pt.

Dados Colectados e Processamento
A aplicação recolhe apenas os dados necessários ao cálculo da estimativa de poupança no IMI, que incluem o valor patrimonial tributário, as áreas do imóvel, os coeficientes de cálculo, as datas de registo e avaliação do imóvel, e os códigos de distrito, concelho e freguesia. Para assegurar a privacidade do utilizador, o site usa expressões regulares (regex) para extrair apenas estes parâmetros do PDF da caderneta predial, ignorando quaisquer dados pessoais presentes no documento. Os dados também podem ser inseridos manualmente pelo utilizador, caso ele prefira esta opção.

Finalidade e Base Jurídica
Os dados são processados unicamente para a finalidade de calcular uma estimativa indicativa de poupança no IMI, com base em informações públicas e privadas fornecidas voluntariamente pelo utilizador. A base jurídica para este tratamento é o consentimento do utilizador, que concorda com o uso do serviço e o fornecimento de informações necessárias.

Medidas de Segurança
São aplicadas medidas técnicas e organizacionais para proteger os dados processados contra acesso não autorizado, uso indevido ou perda. Nenhum dado é armazenado, e as informações são descartadas imediatamente após a conclusão do cálculo.

Período de Conservação dos Dados
Não existe conservação de dados, uma vez que todos os dados fornecidos são eliminados automaticamente após o processamento.

Transferência e Comunicação de Dados para Terceiros
Os dados inseridos não são partilhados, comunicados a terceiros, ou transferidos para fora da União Europeia.

Direitos do Utilizador
Como não há armazenamento de dados pessoais, os direitos de acesso, rectificação, cancelamento, limitação, portabilidade e oposição não se aplicam neste serviço. O utilizador tem a garantia de que todos os dados fornecidos são utilizados exclusivamente para o cálculo imediato e descartados automaticamente após o processamento.
